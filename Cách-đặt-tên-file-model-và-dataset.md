# Đặt tên file model 

Đề xuất đặt tên theo `task-id_model-name_timestamp`

Ví dụ:

* `wt_crf_2018_05_06.tar.gz`
* `tc_naive-bayes_2019_03_03.tar.gz`

Danh sách các task id 

task | id | chú thích
-- | -- | --
sent_tokenize | st |
word_tokenize | wt |
pos_tag | pos |
chunk | chunk | 
ner | ner | 
classify | tc | text classify
sentiment | se
