### Cách đặt tên commit

* Đặt tên commit với prefix `GH-[ISSUE_NUMBER]: nội dung` trỏ đến issue tương ứng
  * ví dụ: `GH-206: fix flake8`

### Cách đặt tên branch

* Tên branch để phát hành có dạng `release-[VERSION]`
  * ví dụ: `release-1.1.9`
